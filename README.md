O que é o GitHub

# O GitHub:

O GitHub é uma plataforma de hospedagem na nuvem para projetos de software que utiliza o sistema Git para controle de versão. Ele permite que desenvolvedores armazenem, gerenciem, compartilhem e colaborem em códigos-fonte de qualquer lugar, funcionando como uma rede social de códigos. É amplamente utilizado para trabalho colaborativo e hospedagem de projetos de código aberto (open source0

# Git
O Git é um sistema de controle de versão distribuído, gratuito e de código aberto, utilizado para rastrear mudanças no código-fonte durante o desenvolvimento de software. Ele permite salvar o histórico de alterações, restaurar versões anteriores e facilita o trabalho colaborativo, sendo instalado localmente na máquina do desenvolvedor

 git init (iniciar repositório);
 git clone (copiar repositório);
 git add (preparar arquivos);
 git commit  (salvar versão);
 git push (enviar para nuvem);
 git pull (atualizar local);
